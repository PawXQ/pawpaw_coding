在 ASCII 編碼中, 數字字符 0~9 在 ASCII 中範圍是 48~57, 透過 char[input] 可以看到各個字符的 ASCII 編碼

除了數字字符, ASCII 編碼還包含了其他字符, 如字母 a~z, A~Z 等等以及各種符號

IntTryParse Method

雖然我們傳入的是 12, 但對於電腦來說, 並不會看到12, 實際上會被分別處理為 1 和 2, 而 1 和 2 經過 char 類型轉換後, ASCII 值分別為 49 和 50, 為了得到我們常見的數值(1 和 2), 需要將這些 ASCII 值減去 48 後進行計算, 才會是我們一般所看到的數值

static bool IntTryParse(string input, out int result)
{
    result = 0;
    for (int i = 0; i < input.Length; i++)
    {
        char c = input[i];
        if (c > 57 || c < 48) // 當 char 轉換後不在範圍內表示輸入的不是數字, 就 return false
        {
            result = 0;
            return false;
        }
        result = result * 10 + (input[i] - 48); // 將之前的 result 擴展一位, 為新加的數字留出空間(例如, 將 12 轉換為 120, 然後加上 3 變成 123)
    }
    return true;
}
