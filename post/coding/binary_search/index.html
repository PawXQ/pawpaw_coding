二分搜尋

Topic

二分搜尋法(binary search)是一種高效且常用的搜尋演算法, 特別適合在已排序的數列中查找特定 element 的位置

這個方法的核心在於每次將搜索範圍縮小一半, 從而迅速找到目標

運作方式：

一開始先找到數列的中間位置, 並將中間的元素與目標值進行比較, 根據比較結果, 有三種可能：

  • 找到目標值(middle = target)： 如果中間位置的元素正好和目標值一樣, 那麼搜索結束, 返回這個元素的位置

  • 目標值較小(middle < target)： 往左邊找. 如果 target 比中間位置的元素小, 表示 target 應該在數列的左半邊. 將搜索範圍縮小到左半邊, 繼續比較

  • 目標值較大(middle > target)： 往右邊找. 如果 target 比中間位置的元素大, 表示 target 應該在數列的右半邊. 將搜索範圍縮小到右半邊, 繼續比較

每次比較後, 搜索範圍會縮小一半, 不斷重複, 直到找到 target, 或確認 target 不在數列中。

namespace Test2
{
    internal class Program
    {
        static void Main(string[] args)
        {
            int[] numbers = { 7, 17, 20, 26, 27, 35, 44, 48, 56, 87 };

            int target = 20;
            int index = Dichotomy(numbers, target);

            Console.WriteLine($"{target} index is {index}.");

            Console.ReadKey();
        }

        static int Dichotomy(int[] number, int target)
        {
            // 初始化搜尋範圍的起點(begin)和終點(end)
            int begin = 0;
            int end = number.Length - 1;

            // 當搜尋範圍內仍有元素時繼續搜尋
            while (begin <= end)
            {
                // 計算中間 index
                int index = (begin + end) / 2;
                if (number[index] < target) // 如果中間的元素小於 target, 將搜尋範圍縮小到右半邊
                {
                    begin = index + 1; // 將起點移動到 index 的右邊一個位置(排除當前的 index)
                }
                else if (number[index] > target) // 如果中間的元素大於 target，將搜尋範圍縮小到左半邊
                {
                    end = index - 1; // 將終點移動到 index 的左邊一個位置(排除當前的 index)
                }
                else if (number[index] == target) // 如果找到 target, 返回 index
                {
                    return index;
                }
            }
            return -1; // 如果搜尋結束仍未找到 target, 返回 -1 表示未找到
        }
    }
}

def binary_search(numbers, target):
    begin = 0
    end = len(numbers)
    while (begin <= end):
        index = (begin + end) // 2
        if numbers[index] > target:
            end = index - 1
        elif numbers[index] < target:
            begin = index + 1
        else:
            return index
    return -1
