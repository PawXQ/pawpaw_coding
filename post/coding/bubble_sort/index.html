[演算法] 氣泡排序法(Bubble Sort)

Bubble Sort

氣泡排序法(bubble sort)又稱為交換排序法, 是一種簡單但效率比較低的排序演算法

原理很簡單, 就是從數列的左側開始, 逐一比較相鄰的兩的 element, 如果相鄰的兩個 element 順序錯誤, 則交換他們的位置, 反之則不動

以數字數列來說, 每次遍歷後，最大的數字會被移動到數列的最右邊, 在經過最多總 element 個數的遍歷次數後, 數列則會從小到大完整排序

但是因為氣泡排序法需要多次的遍歷和交換, 最多需要進行與 element 數量相同次數的遍歷, 因此會需要花費很多時間, 因此不適合處理大量資料

namespace bubble_sort
{
    internal class Program
    {
        static void Main(string[] args)
        {
            int[] numbers = { 26, 44, 56, 87, 48, 27, 20, 7, 35, 17 };

            // bubble
            numbers = BubbleSort(numbers);

            foreach (int item in numbers)
            {
                Console.Write(item + "\t"); // 7, 17, 20, 26, 27, 35, 44, 48, 56, 87
            }

            Console.ReadKey();
        }
        
        static int[] BubbleSort(int[] numbers)
        {
            bool exchange = true;
            for (int i = 0; i < numbers.Length && exchange; i++) // 外層循環控制遍歷次數, 當未發生交換時停止循環, 最多為 element 的個數
            {
                exchange = false;
                for (int j = 0; j < numbers.Length - 1; j++)
                {
                    if (numbers[j] > numbers[j + 1])
                    {
                        (numbers[j], numbers[j + 1]) = (numbers[j + 1], numbers[j]); // 交換兩個 element 的順系
                        exchange = true; // 標記發生了交換, 以便繼續下一次循環
                    }

                }
            }
            return numbers;
        }
    }
}

def bubble_sort(numbers):
    n = len(numbers)
    exchange = True
    for i in range(n):
        if not exchange:
            break
        exchange = False
        for j in range(n - 1):
            if numbers[j] > numbers[j + 1]:
                # 交換兩個元素
                numbers[j], numbers[j + 1] = numbers[j + 1], numbers[j]
                exchange = True
    return numbers


# main program
numbers = [26, 44, 56, 87, 48, 27, 20, 7, 35, 17]

# 排序
numbers = bubble_sort(numbers)

# 輸出結果
for item in numbers:
    print(item, end="\t")
