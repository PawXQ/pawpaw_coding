背包問題有大約下列幾種

  1. 0 1 背包

    • 一個背包

    • 每個物品只有一個, 選或不選

  2. 完全背包

    • 一個背包

    • 每個物品有無數個, 不選或選複數個

  3. 多重背包

    • 多個背包

    • 每個物品有一定數量的複本，可以選擇放入多個背包中

  4. 分組背包

    • 一個背包

    • 物品被分成若干組，每組物品只能選擇一個放入背包

namespace <namespace_name>
{
    internal class Program
    {
        static void Main(string[] args)
        {
            //int[] value = { 4500, 5700, 2250, 1100, 6700 };
            // int[] value = { 3, 4, 5, 6 };
            int[] value = { 1, 5, 2, 4 };
            //int[] weight = { 4, 5, 2, 1, 6 };
            int[] weight = { 2, 3, 5, 7 };
            // int total = 8;
            int total = 10;

            int[,] numbers = new int[weight.Length + 1, total + 1];

            for (int i = 1; i <= weight.Length; i++)
            {
                for (int j = 0; j <= total; j++)
                {
                    if (weight[i - 1] > j)
                    {
                        int origin_value = numbers[i - 1, j];
                        numbers[i, j] = origin_value;
                    }
                    else
                    {
                        int origin_value = numbers[i - 1, j];
                        int new_value = numbers[i - 1, j - weight[i - 1]] + value[i - 1];
                        if (new_value > origin_value)
                        {
                            numbers[i, j] = new_value;
                        }
                        else
                        {
                            numbers[i, j] = origin_value;
                        }
                    }
                }
            }
            Console.WriteLine(weight.Length);
            Console.WriteLine(total);

            Console.WriteLine("Maximum value: " + numbers[weight.Length, total]);
            Console.ReadKey();
        }
    }
}

row i 表示第幾個物品

column j 表示背包的重量

從這個表格可以看到在在 dp[i,j], 表示在第 i 個物品, 在重量 j 的情況下所能獲取的最大價值

藍色的位置是這表格中關鍵得狀態轉換的位置

dp[1,2] => 在最大承重 2 的情況下, 取了第一個物品, value 為 1

dp[2,3] => 在最大承重 3 的情況下, 取了第一個物品, value 為 1

dp[2,5] => 在最大承重 5 的情況下, 取了第一個物品, value 為 1
