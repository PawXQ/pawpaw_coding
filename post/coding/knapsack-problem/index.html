Topic

背包問題有大約下列幾種

  1. 0 1 背包

    • 一個背包

    • 每個物品只有一個, 選或不選

  2. 完全背包

    • 一個背包

    • 每個物品有無數個, 不選或選複數個

  3. 多重背包

    • 多個背包

    • 每個物品有一定數量的複本, 可以選擇放入多個背包中

  4. 分組背包

    • 一個背包

    • 物品被分成若干組, 每組物品只能選擇一個放入背包

主要思考

在動態規劃中, 我們只需要取該重量下最大的價值的組合, 而不需要取得該重量下的所有可能組合

物品相對應的 weight：2, 3, 5, 7

物品相對應的 value：1, 5, 2, 4

最大重量 total： 10

只有一個背包, 物品只有一個, 只能取或不取

namespace <namespace_name>
{
    internal class Program
    {
        static void Main(string[] args)
        {
            //int[] value = { 4500, 5700, 2250, 1100, 6700 };
            // int[] value = { 3, 4, 5, 6 };
            int[] value = { 1, 5, 2, 4 }; // 定義物品的 value
            //int[] weight = { 4, 5, 2, 1, 6 };
            int[] weight = { 2, 3, 5, 7 }; // 定義物品的 weight
            // int total = 8;
            int total = 10; // 定義背包的 total weight

            int[,] numbers = new int[weight.Length + 1, total + 1]; // 使用二維 aarray 來存放動態規劃的結果, 會都 +1 是為了存放不取物品以及當重量為0的情況, 方便之後動態轉移使用

            for (int i = 1; i <= weight.Length; i++) // loop 每一個物品
            {
                for (int j = 0; j <= total; j++) // loop 每一個重量
                {
                    if (weight[i - 1] > j) // 如果當前物品的重量超過當前的可用重量, 不能選取此物品
                    {
                        // 當前物品不能放入背包, 直接繼承前一個物品在當前容量下的最大價值
                        int origin_value = numbers[i - 1, j];
                        numbers[i, j] = origin_value;
                    }
                    else
                    {
                        // 當前物品可以放入背包
                        int origin_value = numbers[i - 1, j]; // 取得前一個物品在目前容量下的最大價值
                        int new_value = numbers[i - 1, j - weight[i - 1]] + value[i - 1]; // 計算放入當前物品后的最大價值
                        if (new_value > origin_value)
                        {
                            numbers[i, j] = new_value;
                        }
                        else
                        {
                            numbers[i, j] = origin_value;
                        }
                    }
                }
            }
            Console.WriteLine(weight.Length);
            Console.WriteLine(total);

            Console.WriteLine("Maximum value: " + numbers[weight.Length, total]);
            Console.ReadKey();
        }
    }
}

狀態轉移思考

在每一個重量情況下, 最大價值是通過比較不考慮當前物品的最大價值和考慮當前物品後所能獲得的最大價值來決定的

dp[i][j] 表示前 i 個物品, 在重量 j 的情況下, 所能取到的最大價值

如果取第 i 個物品, 價值為 dp[i-1][j-weight[i]] + value[i]

如果不取第 i 個物品, 價值為 dp[i-1][j]

狀態轉移： dp[i][j] = max(dp[i-1][j-weight[i]] + value[i], dp[i-1][j])

我們用 dp[2,3] 和 dp[2,5]來說明上述狀態轉移的想法

第二個物品 weight 為 3, value 為 5

  1. 在 dp[2,3] 的時候

    • 不取第二個物品：在重量為 3 時, 不取第二個物品的最大價值為 dp[2-1,3], 即不考慮目前物品時的最大價值, 這裡為 1

    • 取第二個物品：在重量為 3 時，選擇取第二個物品, 此時的最大價值為 dp[2-1, 3-3] + 5, 即在去掉當前物品的重量後的剩餘重量(0)下的最大價值加上當前物品的價值, 具體計算為 dp[1, 0] + 5 = 5

    • 比較與選擇：比較取與不取目前物品時的最大價值, 取較大值為 dp[2,3] 的值, 在這個例子中, dp[2,3] = 5

  2. 在 dp[2,5] 的時候

    • 不取第二個物品：在重量為 5 時, 不取第二個物品的最大價值為 dp[2-1,5], 即不考慮目前物品時的最大價值, 這裡為 1

    • 取第二個物品：在重量為 5 時，選擇取第二個物品, 此時的最大價值為 dp[2-1, 5-3] + 5, 即在去掉當前物品的重量後的剩餘重量(2)下的最大價值加上當前物品的價值, 具體計算為 dp[1, 2] + 5 = 6

    • 比較與選擇：比較取與不取目前物品時的最大價值, 取較大值為 dp[2,5] 的值, 在這個例子中, dp[2,5] = 6

兩個一維 array (滾動數組優化)

現在要來優化剛剛的二維 array, 變成只需要用兩個一維 array 就可以處理

在狀態轉移的思考的過程中, 可以注意到在前 i 個物品所能取到的最大 value, 只和取前 i-1 個物品所能取到的最大 value 有關

因此在過程中不需要紀錄 i-2, i-3, i-n…的狀態

以剛剛的例子 dp[2,3] 和 dp[2,5] 為例：

  1. dp[2,3] 只和 dp[1,3] 和 dp[1,0] 有關

  2. dp[2,5] 只和 dp[1,5] 和 dp[1,2] 有關

dp[i][j] = max(dp[i-1][j-weight[i]] + value[i], dp[i-1][j])

優化成

dp[i][j] = max(dp[0][j-weight[i]] + value[i], dp[0][j])

因為用了兩個一維 array, 因此最後要將 curr array 複製給 prev array

        if (weights[i - 1] > j) {
            curr[j] = prev[j];
        } else {
            curr[j] = Math.Max(prev[j], prev[j - weights[i - 1]] + values[i - 1]);

    // 複製 prev 和 curr

單個一維 array

現在要來優化剛剛的二維 array 和使用兩個 一維 array, 最終只需要使用一個 一維 array 就可以解題

在狀態轉移的思考的過程中, 可以注意到在前 i 個物品所能取到的最大 value, 只和取前 i-1 個物品所能取到的最大 value 有關

因此在過程中不需要紀錄 i-2, i-3, i-n…的狀態

以剛剛的例子 dp[2,3] 和 dp[2,5] 為例：

  1. dp[2,3] 只和 dp[1,3] 和 dp[1,0] 有關

  2. dp[2,5] 只和 dp[1,5] 和 dp[1,2] 有關

dp[j][j] = max(dp[0][j-weight[i]] + value[i], dp[0][j])

當前 j 只和上一層的 j, 以及上一層的j-weight[i] 有關

dp[j] = max(dp[j-weight[i]] + value[i], dp[j]) (這邊注意一定要倒敘)

必須從大到小倒序遍歷 j, 確保在計算當前 dp[j] 時, dp[j - weights[i]] 仍然是原本的值, 這樣可以避免在同一層次的迭代中覆蓋掉所需的數據, 從而保證狀態轉移的正確性

namespace <namespace_name>
{
    internal class Program
    {
        static void Main(string[] args)
        {
            // int[] value = { 3, 4, 5, 6 };
            int[] value = { 1, 5, 2, 4 };
            int[] weight = { 2, 3, 5, 7 };
            // int[] weight = { 2, 3, 4, 5 };
            int total = 10;

            int[] numbers = new int[total + 1]; // 用來存放每個重量下的最大價值, 包含0

            for (int i = 0; i < weight.Length; i++) // loop 每個物品
            {
                for (int j = total; j > 0; j--) // loop 每個重量, 注意由大到小
                {
                    if (weight[i] > j) // 如果當前物品的重量大於當前重量
                    {
                        // 不須更新, 直接保留原來的最大價值
                        int origin_value = numbers[j];
                        numbers[j] = origin_value;
                    }
                    else
                    {
                        int origin_value = numbers[j]; // 當前重量的原最大價值
                        int new_value = numbers[j - weight[i]] + value[i]; // 加入當前物品後的新最大價值
                        if (new_value > origin_value)
                        {
                            numbers[j] = new_value;
                        }
                        else
                        {
                            numbers[j] = origin_value;
                        }
                    }
                }
            }
            Console.WriteLine("Maximum value: " + numbers[total]);

            Console.ReadKey();
        }
    }
}

完全背包

只有一個背包, 物品有複數個, 不取或取複數個

所以每個物品能取的最大數量為 total weight / weight

dp[i][j] = max(dp[i][j],dp[i-1][j - k * weight[i]] + k * weight[i])

題目假設為：

物品相對應的 weight：2, 3, 4, 5

物品相對應的 value：30, 50, 100, 200

最大重量 total： 8

倒序

這邊其實可以把完全背包的題目轉換成01背包的題目下去做思考

例如在第一個物品的情況下, 總重 8 / 2 = 4 其實就可以看做4個重量為2的物品, 這樣也是相同變成 01背包, 取或不取的概念

namespace <namespace_name>
{
    internal class Program
    {
        static void Main(string[] args)
        {
            //int[] value = { 30, 50, 200, 200 };
            int[] value = { 30, 50, 100, 200 };
            //int[] value = { 4500, 5700, 2250, 1100, 6700 };

            int[] weight = { 2, 3, 4, 5 };
            //int[] weight = { 4, 5, 2, 1, 6 };

            int total = 8;

            int[] numbers = new int[total + 1];

            for (int i = 0; i < weight.Length; i++)
            {
                int frequency = total / weight[i]; // 計算目前物品在背包中最多可以出現的次數
                for (int k = 0; k < frequency; k++) // loop 當前物品出現的次數
                {
                    for (int j = total; j > 0; j--)
                    {
                        if (weight[i] > j)
                        {
                            int origin_value = numbers[j];
                            numbers[j] = origin_value;
                        }
                        else
                        {
                            int origin_value = numbers[j];
                            int new_value = numbers[j - weight[i]] + value[i];
                            if (new_value > origin_value)
                            {
                                numbers[j] = new_value;
                            }
                            else
                            {
                                numbers[j] = origin_value;
                            }
                        }
                    }
                }
            }
            Console.WriteLine("Maximum value: " + numbers[total]);

            Console.ReadKey();
        }
    }
}

正序

剛剛我們是以倒敘法把完全背包變成01背包的解法, 但其實只要以正序法下去解題, 就不需要多一步計算可放入次數的動作

還記得剛剛在01背包使用一維 array的時候, 為了避免汙染前面的資料因此使用倒序法, 其實就是怕重複使用變成完全背包的狀況, 因此只能倒序

但是現在物品可以重複使用, 因此在此情況下正序法就是可以正常使用

namespace <namespace_name>
{
    internal class Program
    {
        static void Main(string[] args)
        {
            int[] value = { 30, 50, 100, 200 };
            //int[] value = { 4500, 5700, 2250, 1100, 6700 };

            int[] weight = { 2, 3, 4, 5 };
            //int[] weight = { 4, 5, 2, 1, 6 };

            int total = 8;

            int[] numbers = new int[total + 1];


            for (int i = 0; i < weight.Length; i++)
            {
                for (int j = weight[i]; j <= total; j++) // 從小到大 loop 所有的重量
                {
                    // 當目前重量能裝下當前物品時, 嘗試放入當前物品
                    int new_value = numbers[j - weight[i]] + value[i];
                    // 更新最大價值
                    if (new_value > numbers[j])
                    {
                        numbers[j] = new_value;
                    }
                }
            }
            Console.WriteLine("Maximum value: " + numbers[total]);

            Console.ReadKey();
        }
    }
}
