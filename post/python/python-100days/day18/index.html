Module

module 導入方式

一般 import: import module_name

指定 import: from module_name import specific_function_or_class

全部 import: from module_name import * (不推薦, 會不知道哪個 function, class 是哪個 module 來的)

別名 import: import module_name as alias (為 module 使用別名, 方便辨認)

乖乖使用一般和指定就好, 不過體感還是一般好用

colorgram module

colorgram 是一個 Python module, 用來從圖像中提取顏色. 它能夠分析圖像並傳回最常出現的顏色.

使用 pip 安装：

1

pip install colorgram.py

提取顏色

1

colors = colorgram.extract('image.jpg', 6)

  • 1st parameter：圖片路徑

  • 2nd parameter：要提取的顏色數量

顏色物件屬性

每個提取的顏色都是一個 object, 具有以下 attribute：

  1. rgb: RGB 值(紅、綠、藍)
  2. hsl: HSL 值(色相、飽和度、亮度)
  3. proportion: 該顏色在影像中的比例

import random
import turtle

shape = 10

turtle.colormode(255)

screen = turtle.Screen()
screen.bgcolor("black")

timmy = turtle.Turtle()


def DrawShape(num):
    degree = 360/num
    for i in range(num):
        timmy.fd(100)
        timmy.right(degree)


def generate_random_colors():
    r = random.randint(0, 255)
    g = random.randint(0, 255)
    b = random.randint(0, 255)
    color_tuple = (r, g, b)

    return color_tuple


timmy.penup()
timmy.goto(-50, 0)
timmy.pendown()

for i in range(3, shape+1):
    color = generate_random_colors()
    timmy.color(color)
    DrawShape(i)


screen.exitonclick()

import random
import turtle


def generate_random_colors():
    r = random.randint(0, 255)
    g = random.randint(0, 255)
    b = random.randint(0, 255)
    color_tuple = (r, g, b)

    return color_tuple


def RandomDegree():
    degree_list = [0, 90, 180, 270]
    random_degree = random.choice(degree_list)

    return random_degree


move_step = 200

turtle.colormode(255)

screen = turtle.Screen()
screen.bgcolor("black")

timmy = turtle.Turtle()
timmy.speed(10)


for _ in range(move_step + 1):
    color_tuple = generate_random_colors()
    timmy.color(color_tuple)
    degree = RandomDegree()
    timmy.setheading(degree)
    timmy.fd(20)


screen.exitonclick()

import turtle
import random


def generate_random_colors():
    r = random.randint(0, 255)
    g = random.randint(0, 255)
    b = random.randint(0, 255)
    color_tuple = (r, g, b)

    return color_tuple


def RandomDegree():
    degree_list = [0, 90, 180, 270]
    random_degree = random.choice(degree_list)

    return random_degree


def GenerateRandomAngle():
    random_angle = random.randint(1, 360)

    return random_angle


screen = turtle.Screen()
screen.bgcolor("black")

turtle.colormode(255)

timmy = turtle.Turtle()
timmy.speed(0)
angle = GenerateRandomAngle()
for _ in range(angle):
    color = generate_random_colors()
    timmy.color(color)
    angle_step = 360/angle
    timmy.circle(100)
    timmy.right(angle_step)


screen.exitonclick()

import random
import turtle

color_list = [(228, 235, 231), (199, 162, 100), (62, 91, 128), (140, 170, 192), (139, 90, 48), (219, 206, 119), (135, 27, 52), (32, 41, 67), (78, 16, 36), (149, 59, 85), (167, 154, 49), (187, 143, 162), (134, 183, 147),
              (46, 55, 100), (181, 95, 107), (56, 39, 27), (96, 118, 167), (80, 150, 159), (89, 152, 92), (71, 118, 93), (220, 175, 187), (169, 207, 163), (161, 202, 215), (192, 95, 74), (178, 187, 213), (46, 73, 75), (76, 69, 44)]


def random_color():
    return random.choice(color_list)


screen = turtle.Screen()
screen.bgcolor("black")

turtle.colormode(255)

timmy = turtle.Turtle()
timmy.hideturtle()
timmy.speed(0)

for i in range(10):
    timmy.penup()
    timmy.goto(-250, -250 + i * 50)
    for _ in range(10):
        timmy.dot(20, random_color())
        timmy.forward(50)


screen.exitonclick()
