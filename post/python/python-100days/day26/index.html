Day 26 - Intermediate - List Comprehension and the NATA Alphabet

List Compression

Syntax

基本語法

1

new_list = [new_item for item in list]

重組語句: for item in list create new_item

中文翻譯: 從 list 中拿出每個 item 生成 new_item 並加入到 new_list 中

  1. new_list：新 list 的名稱
  2. new_item：對每個 element 進行的操作
  3. item：原始列表中的每個 element
  4. list：原始列表或可迭代的 sequence

Example

1. 基本轉換

將原始列表的每個元素增加 1：

1
2

numbers = [1, 2, 3]
new_numbers = [n + 1 for n in numbers]  # result: [2, 3, 4]

2. 處理字符串

將字符串拆分為字母列表：

1
2

name = "Angela"
letters_list = [letter for letter in name]  # result: ['A', 'n', 'g', 'e', 'l', 'a']

3. 使用 Range

將範圍內的數字翻倍：

1

range_list = [num * 2 for num in range(1, 5)]  # result: [2, 4, 6, 8]

If condition List Compression

基本語法

1

new_list = [new_item for item in list if test]

重組語句: for item in list if test, create new_item

中文翻譯: 從 list 中拿出每個 item, 如果符合 test, 則生成 new_item 並加入到 new_list 中

添加 if condition, fileter element：

1
2
3

names = ["Alex", "Beth", "Caroline", "Dave", "Eleanor", "Freddie"]
short_names = [name for name in names if len(name) <= 4]  # result: ['Alex', 'Beth', 'Dave']
long_names = [name.upper() for name in names if len(name) >= 5]  # result: ['CAROLINE', 'ELEANOR', 'FREDDIE']

Dict Compression

Syntax

基本語法

1

new_dict = {new_key: new_value for item in iterable}

重組語句：for item in iterable, create new_key: new_value

中文翻譯：從 iterable 中取出每個 item, 生成 new_key 和 new_value 作為 key value pair

Example

Create from List

1
2
3
4
5

names = ['Alex', 'Beth', 'Caroline']
students_scores = {student: random.randint(1, 100) for student in names}

# result
# {'Alex': 94, 'Beth': 85, 'Caroline': 21}

Create from Dict

1
2
3
4

passed_students = {student: score for student, score in students_scores.items() if score >= 60}

# result:
# {'Alex': 94, 'Beth': 85}

If condition Dict Compression

基本語法

1

{key: value for key, value in original_dict.items() if condition}

重組語句: for each key, value in original_dict, if condition is met, create key: value

中文翻譯: 從 original_dict 中取得每個 key 和 value, 當符合 condition 時, 生成 key: value

Complex key value convert

1

{key.upper(): value * 2 for key, value in original_dict.items()}

重組語句：for each key, value in original_dict, create key.upper(): value * 2

中文翻譯：從 original_dict 中取得每個 key 和 value, 將 key 轉為大寫並將 value 乘以 2, 生成 key.upper(): value * 2

訪問特定數據

使用 row object 讀取數據

每一行都是一個 Pandas Series object

1
2
3

for index, row in student_data_frame.iterrows():
    print(row.student)  # 訪問 student row
    print(row.score)    # 訪問 score row

條件篩選

可以在迭代過程中進行 condition filter：

1
2
3
4

# 查找特定學生的分數
for index, row in student_data_frame.iterrows():
    if row.student == 'Angela':
        print(row.score)  # result 56

重要概念總結

  1. DataFrame 迭代可以按 column 或 row 進行
  2. .items() 適合按 column loop through
  3. .iterrows() 是按 row loop through 的推薦方式
  4. row object 是 Pandas Series type, 可以使用 row.["column_name"] 取得特定的資料
  5. 可以在迭代過程中進行條件篩選和數據處理
