今天只有題目~~

Coffe Machine

題目要求：

  1. 三種咖啡口味：

    • 濃縮咖啡
    • 拿鐵
    • 卡布奇諾 每種咖啡有不同的配方, 使用不同量的水, 牛奶, 咖啡, 且價格不同

  2. 資源管理：

    • 初始資源：
      • 300 ml 水
      • 200 ml 牛奶
      • 100 g 咖啡
    • 機器必須管理和跟蹤這些資源的消耗情況

  3. 投幣操作：

    • 接受四種類型的硬幣：
      • Penny($0.01)
      • Nickel($0.05)
      • Dime($0.10)
      • Quarter($0.25)
    • 機器需計算投幣總金額, 判斷是否足夠支付所選飲品, 若不足則退款

  4. 打印報告：咖啡機應能夠打印剩餘資源(如水, 牛奶, 咖啡)的報告, 以及收入金額

  5. 檢查資源是否足夠：在製作咖啡之前, 咖啡機需檢查是否有足夠的資源

from menu import MENU

resources = {
    "water": 300,
    "milk": 200,
    "coffee": 100,
    "money": 0,
}


def report():
    for i in resources:
        if i == 'water':
            print(f"{i.capitalize()}: {resources[i]}ml")
        if i == 'milk':
            print(f"{i.capitalize()}: {resources[i]}ml")
        if i == 'coffee':
            print(f"{i.capitalize()}: {resources[i]}g")
        if i == 'money':
            print(f"{i.capitalize()}: ${resources[i]}")


def calculate():
    quarters = int(input('How many quarters: '))
    dimes = int(input('How many dimes: '))
    nickles = int(input('How many nickles: '))
    pennies = int(input('How many pennies: '))
    return 0.25 * quarters + 0.1 * dimes + 0.05 * nickles + 0.01 * pennies


def enough_money(coffee, money_received):
    cost = MENU[coffee]['cost']
    if money_received < cost:
        print("Sorry that's not enough money. Money refunded.")
        return False
    resources['money'] += cost
    print(f"Here is ${round(money_received - MENU[coffee]['cost'], 2)} in change.")
    print(f"Here is your {coffee} ☕️. Enjoy!")
    return True


def enough_resource(coffee):
    ingredients = MENU[coffee]['ingredients']
    for item in ingredients:
        if ingredients[item] > resources[item]:
            print(f"Sorry there is not enough {item}.")
            return False
    return True


def make_coffee(coffee):
    ingredients = MENU[coffee]['ingredients']
    for item in ingredients:
        resources[item] -= ingredients[item]


while True:
    user_input = input("Invalid input. Please type 'espress or 'latte' or 'cappuccino': ").lower()
    while user_input not in ["espresso", "latte", "cappuccino", "off", "report"]:
        user_input = input("Invalid input. Please type 'espress or 'latte' or 'cappuccino': ").lower()
    if user_input == 'report':
        report()
        continue
    resource_check = enough_resource(user_input)
    if not resource_check:
        continue
    print("Please insert coins.")
    payment = calculate()
    if enough_money(user_input, payment):
        make_coffee(user_input)
