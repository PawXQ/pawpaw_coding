Day 2 - Begineer - Understanding Data Types and How to Manupulate Strings

Data type

python 有以下幾個主要的 data type

strings, integer, float , boolean

strings

就是一串連續的字元(chraracter)

使用 "" 或 '' 包裹

可以使用 下標（subscripting） 來提取字串中的特定字元,

index 從 0 開始, "Hello"[0] 可以得到 'H'

也可以從負數開始 "Hello"[-1] 可以得到 'o'

integer

整數

如果數字很大, 可以使用下滑線'_'幫助閱讀, python 會自動忽略, 不影響程式

1_234_567 等於 1234567

float

浮點數是有小數點的數字, 如 3.14159.

boolean

布林值是表示真或假的值, 分別是 True 和 False

type()

檢查放在括號之間的任何內容, 並 return 它的 data type

f-string

f-strings 是 Python 中格式化 strings 的方法, 可以在 strings 中插入不同的 variable 和 data type, 不須手動轉換類型

使用方式：

在 stromgs 前加上 f，然後在字串中用 {} 包圍 variable, 它會將 variable 或表達式計算後的值替換進去

一般未使用 f-string

1
2

height = 1.8
isWinning = True

print("Your score is " + str(score) +", your height is " + str(height) + ", you are winning in " + str(isWinning) + ".")

# Your score is 0, your height is 1.8, you are winning in True.

# Your score is 0, your height is 1.8, you are winning in True.
