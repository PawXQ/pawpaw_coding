For Loop

循環(Loops)的概念：

用於重複執行某些操作

for loop 的基本結構

1
2

for item in list_of_items:
    # Do something to each item

Loop & List

for loop 與 list 一起使用：

  • 可以遍歷 list 中的每個 item

  • 每次循環將 list 中的一個 item 賦值給循環 variable

示例：

1
2
3

fruits = ["apple", "peach", "pear"]
for fruit in fruits:
    print(fruit)

在上述的例子中 fruit 就是循環 variable

range()

功能：

  • 用於生成一系列數字

  • 常與 for loop 一起使用

特點：

  • 生成的範圍包含起始值，不包含結束值

  • 默認 step 為 1

  • 單獨使用時不會輸出任何內容

syantx：

range(start, stop, step)

  • start: 計數從 start 開始, 預設是從 0 開始, 例如 range(5) 等價於 range(0, 5

  • stop: 數到 stop 結束, 但不包含 stop, 例如：range(0, 5) 是 [0, 1, 2, 3, 4] 沒有 5

舉例：

1
2

for number in range(1, 11):
    print(number)  # 輸出 1 到 10

自定義 step

1
2

for number in range(1, 11, 3): # 從 1 開始, 結束是 11, step 是 3
    print(number)  # 輸出 1, 4, 7, 10

    if i % 3 == 0:
        a += "Fizz"
    if i % 5 == 0:
        a += "Buzz"
    if a == "":
        a = i
    print(a)

Password Generator

輸入要求：

功能目標：

兩種版本：

簡單版： 依序產生密碼(先字母, 再符號, 再數字)

passwd_list = []
passwd = ""

for i in range(nr_letters):
    passwd_list.append(random.choice(letters))

for i in range(nr_symbols):
    passwd_list.append(random.choice(numbers))

for i in range(nr_numbers):
    passwd_list.append(random.choice(symbols))

# easy
for i in passwd_list:
    passwd += i

print(passwd)

# hard1
for i in range(len(passwd_list)):
    num = random.randint(0, len(passwd_list)-1)
    passwd += passwd_list[num]
    del passwd_list[num]

print(passwd)

# hard2
random.shuffle(passwd_list)
for i in passwd_list:
    passwd += i

print(passwd)

Hard2 的解法, 是使用 random.shuffle() 的方式, 直接將 list 打亂
