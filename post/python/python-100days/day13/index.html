Day 13 - Begineer - Debugging: How to Find and Fix Errors in your Code

Debug

描述問題

  • 明確知道問題是什麼, 清楚表達目前遇到的狀況

測試假設

  • 明白代碼中的隱含假設, 了解 program 中所使用的 function

  • 檢查這些假設是否正確

重現錯誤

  • 對於間歇性出現的錯誤,嘗試重現問題

  • 找出觸發錯誤的具體條件

  • 修改 code 使錯誤每次都出現, 以便於修正

模擬電腦角色

  • 逐行閱讀code, 模擬電腦執行過程

  • 評估每具 code 的結果

  • 跟隨邏輯流程, 檢查條件判斷

IDE Error

  • 優先修復編 IDE 顯示的錯誤(如 indentation 錯誤)

  • 解決運行時在 console 出現的錯誤

使用try-except block

  • 將可能出錯的代碼放入try塊中

  • 使用except捕獲特定類型的錯誤(如ValueError)

  • 在except塊中提供用戶友好的錯誤訊息和處理方法

def get_user_age():
    while True:
        try:
            age = int(input("Please enter your age: "))
            print(f"Your age is {age}")
            break  # 獲得年齡後跳出
        except ValueError:
            # 捕捉 ValueError 並提供友好的提示
            print("Invalid input. Please enter a valid number.")

get_user_age()

使用 print()

  • 使用 print 作為 debugging 工具

  • 步驟

    • 在關鍵位置添加 print() 語句
    • 使用 f-string output variable

Debugger

  • 設置 breakpoint, 使用 debug run mode

Debugger 主要功能

  • Step Over: 逐行執行 code

  • Step Into: 進入函數定義

  • Step Into My Code：只進入自定義代碼, 忽略庫函數

  • Step Out: 退出當前function
