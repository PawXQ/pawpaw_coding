Day 9 - Begineer - Dictionaries, Nesting and Secret Auction

Dictionary

  • 由 key 和 value pair 組成

  • key： Dictionary 裡面的單字, value： 單字的解釋

  • key 是必要的, value 可以是空值

  • value 可以是任意 data type, 但 key 必須是不可變的, 如 string, int

基本 syntax

{key: value}

多個 key-value pair 用 , 隔開

1
2
3
4

    key1: value1,
    key2: value2

def acution_check(dict):

    for i in dict:

bid_dict = {}

while (True):
    print("Welcome to the secret auction program.")
    name = input("What is your name?: ")
    bid = float(input("What's yout bid?: "))
    bid_dict[name] = bid

    next = input("Are there any other bidders? Type 'yes' or 'no'.\n").lower()
