Function

function 是 code 中用來執行特定功能的部分

Python 有許多 built-in function, 如 len(), int(), print(), range() 等

python3-doc-built-function

function 後面一定會帶 (), 如 print()

While Loop

while loop 會在特定條件為 True 時重複執行

與 for loop 相比, while loop更適合不確定迭代次數的情況

syntax:

1
2

while something_is_true
    # Do something repeatedly

sample：

1
2
3
4

count = 5
while count > 0:
    print(count)
    count -= 1

while vs for

for loop 適用於已知迭代次數或遍歷列表的情況

while loop 適用於基於條件的重複執行

infinite loop

  • 當條件永遠為 True 時,會導致無限迴圈
