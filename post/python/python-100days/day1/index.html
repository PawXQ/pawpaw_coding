Print()

print() 為 python 最基本 show 內容的方式

使用方式：

在 print 之後的 ("") 中輸入要輸出的文字

單引號'' 和雙引號 "" 為了表示內容不是 code, 是要輸出的 string

  • 單行輸出
1

`print("<text>")`
  • 多行輸出, 使用 \n 達到換行, 進行多行輸出
1

`print("<text>\n<text>")`
  • 字串連接 (string concatenation), 透過 + 達成達成 string 的連接
1

`print("Hello" + "world!")`
  • 使用 ,，自動產生空格
1

`print("Hello", "Angela")`
  • 如果句子中要使用雙引號""或單引號''，最外面的引號要使用不同於 text 所使用的, 或使用 \ 進行跳脫
1
2
3

print("This's a pen.")
print('This is a "pen".')
print('This\'s is a pen.')

input()

input() 為用來獲取 user 的輸入

()括號, ""引號裡面是要給 user 看的 prompt

1
2
3

`input("A prompt for the user")`

`input("What is your name?")`
  • 換行輸入, 在最後加上 \n 可以達成讓 user 換行輸入的效果
1

`input("What is your name?\n")`
  • print() + input()
1
2
3

# input() will get user input in console
# Then print() will print the word "Hello" and the user input.
print("Hello " + input("What is your name?"))

Variable

命名規則

  • 不能有空格
  • 開頭不能為數字，中間可以
  • 要做分隔一般使用 _, e.g. user_name = "Angela"

變數設定方式

<variable_name> = value

使用 "", ''： 當 variable 的 value 是 string 時, 需要用引號

不使用 "", ''： 當 variable 的 value 是 num(int, double), bool(True, False), list, dict 等型別時, 不需要使用引號
