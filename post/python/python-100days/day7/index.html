Day 7 - Begineer - Hangman

今天只有題目~~

Hangman

題目要求：

  1. 初始化遊戲：

    • 隨機選擇一個單詞並顯示初始化畫面(例如, 顯示下劃線 _)

  2. 處理猜測：

    • 允許用戶輸入字母
    • 檢查字母是否在單詞中並更新顯示
    • 如果猜錯，血量減少

  3. 顯示遊戲狀態：

    • 根據剩餘的血量顯示不同階段的 Hangman ASCII art

  4. 遊戲結束條件：

    • 如果血量為 0，顯示「遊戲結束！」
    • 如果猜對所有字母，顯示「恭喜，你贏了！」

附註：

  • 防止用戶重複猜測相同字母

import random
import hangman_art
import hangman_words

# word_list = ["aardvark", "babppn", "camel"]
word_list = hangman_words.word_list
answer = random.choice(word_list)
# answer = "abcda"
place_holder = []
# char_already_guess = []

live = 6

print(hangman_art.logo)

for i in range(len(answer)):
    place_holder.append("_")

print(f"answer: {answer}")
print(f"Start with {live} live.")

while True:
    print(''.join(place_holder))
    user_guess = input("Guess a letter: ").lower()
    

    if user_guess in place_holder:
        print(f"You've already gussed {user_guess}.")
        # continue

    # char_already_guess.append(user_guess)

    for i in range(len(answer)):
        if user_guess == answer[i]:
            place_holder[i] = user_guess

    if user_guess not in answer:
        live -= 1
        print("Guess wrong live -1.")

    if live == 0:
        print(f"Live returns to {live}.")
        print(f"Correct answer is \"{answer}\"")
        print("Game Over!")
        print(hangman_art.stages[live])
        break

    if "_" not in place_holder:
        print("You Win!")
        break

    print(f"Live still have {live}.")
    print(hangman_art.stages[live])
