Day 12 - Begineer - Scope & Number Guessing Game

Scope

Scope(作用域), 簡單的理解就是 variable 和 function 的可使用區域範圍

Local Scope

  • 存在於 function 內部

  • 在 function 內定義的 variable 只能在該 function 內使用

e.g.

1
2
3
4
5

pythonCopydef drink_potion():
    potion_strength = 2  # local variable
    print(potion_strength)  # 可以使用

print(potion_strength)  # error: 在function外部無法使用

Global Scope

  • 在最外層(不在任何函數內)定義的 variable

  • 可以在整個文件的任何地方訪問, 包括 function 內部

e.g.

1
2
3
4

player_health = 10  # global variable

def drink_potion():
    print(player_health)  # 可以訪問 global variable

variable 訪問規則

  • local variable 只能在定義它的 function 內使用

  • global variable 可以在任何地方使用, 但在 function 內修改需要特殊聲明(global parameter)

  • nested function 中, 內部 function 可以訪問外部 function 的 variable

Block Scope

  • Python 實際上沒有 Block Scope 的概念

  • function 內的 variable：無論 variable 在哪個 code block 內(如 if、while), 只要是在 function 內定義的, variable 的作用範圍就是整個函數(即局部範圍, local scope)

  • function 外的 variable：如果 code block 不在 function 內, 那麼 variable 將具有全域範圍(global scope), 可在整個 program 中使用

修改 Global Scope variable

Global vs Local 變數

  • Global variable: 在函數外定義, 整個程式都能訪問

  • Local variable: 在函數內定義, 只在函數內可用

  • 同名的 Global 和 Local 變數是完全不同的變數

  • Global constants 是全域 variable 的一種, 通常定義後不會再更改, 通常使用全大寫字母，單字間以**底線(underscore)**分隔

使用 global parameter

enemies = 1

def increase_enemies():
    global enemies
    enemies += 1
    print(f"enemies inside function: {enemies}")

increase_enemies()
print(f"enemies outside function: {enemies}")

import random

answer = random.randint(1, 100)
print(f"answer: {answer}")

EASY_LEVEL = 10
HARD_LEVEL = 5

print("Welcom to the Number Guesss Game!")
print("I'm thinking a number between 1 and 100")

difficulty = input("Choose a difficulty. Type 'easy' or 'hard': ")

if difficulty == "easy":
    attempts = EASY_LEVEL
if difficulty == "hard":
    attempts = HARD_LEVEL


while (True):
    print(f"You have {attempts} attempts remaining to guess the number.")
    guess = int(input("Make a guess: "))
    attempts -= 1
    if guess > answer:
        print("Too High.")
    if guess < answer:
        print("Too Low.")
    if guess == answer:
        print(f"You got it! The answer was {answer}")
        break
    if attempts == 0:
        print("You've run out of guessus, you lose.")
        break
