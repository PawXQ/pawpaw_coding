Control Flow

程式中控制執行順序的結構, 根據特定條件, 執行特定動作

if / else

Syntax format:

關鍵字 if 然後是 測試條件 再來是 :, :之後是 縮進的 code block

如果條件滿足則被執行

如果條件不滿足跳到, 關鍵字 else 帶 :, 執行底下的縮進的code block

    do this
else:
    do this

    bill = 5
