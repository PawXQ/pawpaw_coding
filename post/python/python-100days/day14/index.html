今天只有題目~~

Higher Lower Game

題目要求：

  1. 遊戲目標：玩家需要猜測 Instagram 上兩個帳號中誰的追蹤者更多

  2. 基礎數據：

  • 使用 game_data.py 檔案中的列表, 每個元素是一個包含以下key的dict：
  • name: 帳號名稱
  • follower_count: 追蹤者數量(以百萬為單位)
  • description: 帳號描述
  • country: 帳號所屬國家
  1. 遊戲流程：
  • 顯示兩個待比較的帳戶, 玩家需要猜測哪個帳號的追蹤者更多
  • 若猜對, 繼續遊戲, 先前的 B 成為新的 A, 系統再產生一個新帳號作為 B
  • 若猜錯, 遊戲結束, 並顯示得分

import os
import random
from game_data import data
import art


def choose_data(data: list):
    len_game_data = len(data)
    choose = (random.choice(range(len_game_data)))
    return data[choose]


def compare_followers(a_follower, b_follower):
    if a_follower > b_follower:
        return 'a'
    else:
        return 'b'


def context(a_ccount, b_account):
    print(f"Compare A: {a_ccount['name']}, {a_ccount['description']}, from {a_ccount['country']}.")
    print(art.vs)
    print(f"Compare B: {b_account['name']}, {b_account['description']}, from {b_account['country']}.")


def check_invalid_comparison(a_account, b_account):
    a_account_follower = a_account['follower_count']
    b_account_follower = b_account['follower_count']
    if a_account != b_account or a_account_follower != b_account_follower:
        return True


def get_valid_b_account(a_account, data):
    while True:
        b_account = choose_data(data)
        if check_invalid_comparison(a_account, b_account):
            return b_account


def Game():
    score = 0
    a_account = choose_data(data)

    while True:
        b_account = get_valid_b_account(a_account, data)

        result = compare_followers(a_account['follower_count'], b_account['follower_count'])
        print(result)

        if score > 0:
            print(f"You're right! Current score: {score}.")

        context(a_account, b_account)

        guess = input(f"Who has more followers? Type 'A' or 'B': ").lower()

        if guess == result:
            score += 1
            a_account = b_account
        else:
            print(f"Sorry, that's wrong. Final score: {score}")
            break
        os.system("cls")
