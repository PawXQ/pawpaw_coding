Day 33 - Intermediate+ - API Endpoints & API Parameters - ISS Overhead Notifier

API Endpoint

API endpoint 是一個用來存取資料的特定 URL 位置, 就是說丟 request 給他的時候, 可以return 相對應的資料

  • api.coinbase.com - Coinbase的 API endpoint
  • http://api.open-notify.org/iss-now.json - ISS位置的 API endpoint

HTTP status code

當開啟網頁或和 url 座溝通的時候會有一個 status code, 來表示和網頁溝通的結果

透過第一個數字, 可以快速理解回傳的狀態

  • 1xx: 處理中,尚未完成
  • 2xx: 請求成功
  • 3xx: 無權限存取
  • 4xx: 客戶端錯誤(如404表示資源不存在)
  • 5xx: 伺服器端錯誤

API Parameters vs Request Body

HTTP Methods 與資料傳送方式

  • GET 請求：通常使用 URL parameters
  • POST/PUT/PATCH 請求：通常使用 request body

URL Parameters

  • 加在 URL 後面
  • 適合小量、簡單的資料
  • 會顯示在 URL 中
  • 常用於查詢條件

parameters = {

Request Body

  • 放在 HTTP 請求主體中
  • 適合大量或複雜的資料
  • 不會顯示在 URL 中
  • 常用於傳送表單或 JSON 資料

簡單判斷法則：

  1. 用 Parameters 當：

    • 你在「查詢」或「過濾」資料
    • 資料量小且簡單
    • 使用 GET 請求
    • 希望 URL 可以被分享或書籤

  2. 用 Data/Body 當：

    • 你在「創建」或「修改」資料
    • 傳送複雜的資料結構
    • 傳送敏感資訊
    • 資料量較大
    • 使用 POST/PUT/PATCH 請求

基本上可以想成: 如果你是在「問問題」用 Parameters, 如果你是在「提供資訊」用 Data

import requests
from datetime import datetime
import smtplib
import time

USERNAME = "asd246836@gmail.com"
PASSWORD = "jnay jpna vkui arki"

MY_LAT = 25.032969  # Your latitude
MY_LONG = 121.565414  # Your longitude


# 檢查目前太空站位置
def get_current_iss_position():
    response = requests.get(url="http://api.open-notify.org/iss-now.json")
    response.raise_for_status()
    data = response.json()

    iss_latitude = float(data["iss_position"]["latitude"])
    iss_longitude = float(data["iss_position"]["longitude"])
    return iss_latitude, iss_longitude

# Your position is within +5 or -5 degrees of the ISS position.


# 檢查目前是否為夜間
def is_night():
    parameters = {
        "lat": MY_LAT,
        "lng": MY_LONG,
        "formatted": 0,
    }

    response = requests.get("https://api.sunrise-sunset.org/json", params=parameters)
    response.raise_for_status()
    data = response.json()
    sunrise = int(data["results"]["sunrise"].split("T")[1].split(":")[0])
    sunset = int(data["results"]["sunset"].split("T")[1].split(":")[0])

    time_now = datetime.now()
    now_hour = time_now.hour
    if (sunrise < now_hour < sunset):
        return False
    return True


# 寄送 email
def send_email(send_address, receive_addr, password):
    with smtplib.SMTP("smtp.gmail.com") as connection:
        connection.starttls()
        connection.login(send_address, password=password)
        connection.sendmail(
            from_addr=send_address,
            to_addrs=receive_addr,
            msg=f"Subject: ISIS through over head!\n\nLook up! The ISS is above you in the sky."
        )


# 判斷太空站是否越過頭頂
def is_iss_overhead():
    iss_lat, iss_long = get_current_iss_position()
    if (MY_LAT - 5 <= iss_lat <= MY_LAT + 5) and (MY_LONG - 5 <= iss_long <= MY_LONG + 5):
        return True
    return False


# If the ISS is close to my current position
# and it is currently dark
# Then send me an email to tell me to look up.
# BONUS: run the code every 60 seconds.

while True:
    if is_iss_overhead() and is_night():
        send_email(send_address=USERNAME, receive_addr=USERNAME, password=PASSWORD)

    time.sleep(60)
