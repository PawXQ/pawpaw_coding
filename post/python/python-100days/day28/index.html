Method

Canvas Methods:

create image x,y 是表示圖片的中心位置

  • create_image(x, y, image=img)

在 canvas widget 上設定 text

  • create_text(x, y, text="text", fill="color", font=(name, size, "style"))

highlightthickness 表示 Canvas 邊框

  • canvas = Canvas(width=200, height=224, bg=YELLOW, highlightthickness=0)

PhotoImage:

讓 tkinter 讀取外部的圖片檔, 讓之後的組件可以使用該圖片

  • tomato_img = PhotoImage(file="tomato.png")

Canvas 使用 image

  • canvas.create_image(x, y, image=tomato_img)

Label 使用 image

  • label = Label(image=tomato_img)
  • label.pack()

Event Driven

tkinter-after()

after — Execute a command after a time delay

tkinter GUI 程式是事件驅動的

所以到計時等內容無法透過一般 import timer 然後 sleep 來操作

tkinter GUI 透過主循環(mainloop)持續監聽事件, 因此需要透過 after(ms, function, *args)：延遲執行指定函數

在這裡的 *arg 都會傳入前面使用的 function

  • after(ms, function, *args): 延遲執行指定函數

Canvas Methods:

Strongly Typing / Dynamic Typing

核心概念

Python 是強類型但動態類型的語言, 允許變數在運行時改變資料類型

Dynamic Typing 特性

  • 強類型：嚴格檢查型別錯誤
  • 動態型別：變數可以改變資料類型
  • 靈活性：允許在運行時轉換資料類型

類型系統特點

  1. 強類型檢查：
1
2

3 + "4"  # TypeError
x ** "3"  # TypeError
  1. 動態類型：
1
2

a = 3        # type: int
a = "Hello"  # type: str 允許改變

import tkinter
# ---------------------------- CONSTANTS ------------------------------- #
PINK = "#e2979c"
RED = "#e7305b"
GREEN = "#9bdeac"
YELLOW = "#f7f5dd"
FONT_NAME = "Courier"
WORK_MIN = 25
SHORT_BREAK_MIN = 5
LONG_BREAK_MIN = 20
reps = 4
marks = ""
timer = None
is_timer_running = False  # 用於防止重複啟動計時器


# ---------------------------- TIMER RESET ------------------------------- #

def reset_timer():
    window.after_cancel(timer)
    label_timer.config(text="Timer", fg=GREEN)
    canvas.itemconfig(timer_text, text="00:00")
    label_check.config(text="")
    global reps
    reps = 4

# ---------------------------- TIMER MECHANISM ------------------------------- #


def start_timer():
    global reps, is_timer_running

    # 防止重複啟動計時器
    if is_timer_running:
        return

    is_timer_running = True
    if reps == 0:
        # 計時完成後停止
        label_timer.config(text="Done!", fg=RED)
        canvas.itemconfig(timer_text, text="00:00")
        is_timer_running = False
        return

    reps -= 1
    work_sec = WORK_MIN * 60
    short_break_sec = SHORT_BREAK_MIN * 60
    long_break_sec = LONG_BREAK_MIN * 60

    if reps == 0:
        label_timer.config(text="Break", fg=RED)
        count_down(8)
    elif reps % 2 == 1:
        label_timer.config(text="Work", fg=GREEN)
        count_down(5)
    elif reps % 2 == 0:
        label_timer.config(text="Break", fg=PINK)
        count_down(2)


# ---------------------------- COUNTDOWN MECHANISM ------------------------------- #


def count_down(count):
    global timer, is_timer_running

    count_min = count // 60
    count_sec = count % 60

    if count_sec <= 10:
        count_sec = f"0{count_sec}"

    canvas.itemconfig(timer_text, text=f"{count_min}:{count_sec}")
    if count >= 0:
        timer = window.after(500, count_down, count - 1)
    else:
        if reps % 2 == 1:
            global marks
            marks += "✔"
            label_check.config(text=marks)
        is_timer_running = False  # 倒數完成後允許重新啟動
        start_timer()


# ---------------------------- UI SETUP ------------------------------- #
window = tkinter.Tk()
window.configure(bg="gray")
window.title("Pomodoro")
window.config(padx=100, pady=100)


pomodoro_image = tkinter.PhotoImage(file="pomodoro-start/tomato.png")

canvas = tkinter.Canvas(width=200, height=223, bg="gray", highlightthickness=0)
canvas.create_image(100, 112, image=pomodoro_image)
timer_text = canvas.create_text(100, 130, text="00:00", fill="white", font=(FONT_NAME, 30, "bold"))
canvas.grid(column=2, row=1)


label_timer = tkinter.Label(text="Timer", font=(FONT_NAME, 40, "bold"), fg=GREEN, bg="gray")
label_timer.grid(column=2, row=0)

label_check = tkinter.Label(font=(FONT_NAME), bg="gray", fg=GREEN)
label_check.grid(column=2, row=3)

buttom_start = tkinter.Button(text="Start", command=start_timer)
buttom_start.grid(column=1, row=2)

buttom_reset = tkinter.Button(text="Reset", command=reset_timer)
buttom_reset.grid(column=3, row=2)

window.mainloop()
