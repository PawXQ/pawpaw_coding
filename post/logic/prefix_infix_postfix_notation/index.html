前、中、後序簡易說明

底下的內容是在看過上面的文章整理而成, 並引用相關內容


這次的題目為數學計算的中序轉後序

什麼是前中後序!? 就是電腦計算數學的方式

在一般我們的四則運算中, 有「先乘除, 後加減」的規則, 但是對於電腦來說, 並沒有這種優先順序的概念

前序只要從右邊算到左邊, 而後序更只要單純從左邊算到右邊就好

前, 中, 後序的主要差別是在運算子(operator, 如 +, -, *, /) 和運算元(operand, 如 1, 2, -3)的放置位置差異

就以「1加2」 舉例

中序：1 + 2

前序：+ 1 2

後序：1 2 +

順序整理如下：

前序：運算子 運算元 運算元

中序：運算元 運算子 運算元

後序：運算元 運算元 運算子

前序式、中序式、後序式的運算方式

前序式的運算

e.g.

- + 2 * 3 1 9

依照上面的規則, 先找到兩個運算元, 和一個運算子, 並且敘述方向為左到右, 雖然計算順序是右到左

  1. 找到 * 3 1, 念作 「3乘1」等於 3 之後替換掉前序的 * 3 1 => - + 2 3 9

  2. 找到 + 2 3, 念作 「2加3」等於 5 之後替換掉前序的 + 2 3 => - 5 9

  3. 找到 - 5 9, 念作 「5減9」等於 -4, 最後的答案就是 (-4)

中序式的運算

e.g.

2 * (3 + 1) - 5 * 3

從左邊往右邊看, 從括號內往括號外看, 且先乘除後加減

相信不用解釋最後就是 (-7)

後序式的運算

e.g.

2 3 1 * + 9 -

依照上面的規則, 先找到兩個運算元, 和一個運算子, 並且敘述方向為左到右, 計算順序也是左到右

  1. 找到 3 1 *, 念作 「3乘1」等於 3 之後替換掉前序的 3 1 * => 2 3 + 9 -

  2. 找到 2 3 +, 念作 「2加3」等於 5 之後替換掉前序的 2 3 + => 5 9 -

  3. 找到 5 9 -, 念作 「5減9」等於 -4, 最後的答案就是 (-4)

namespace <namespace_name>
{
    internal class Program
    {
        static void Main(string[] args)
        {
            string input = "-+2*319";
            int result = 0;

            Stack<int> stack = new Stack<int>();

            for (int i = input.Length - 1; i >= 0; i--)
            {
                char c = input[i];
                if (!Char.IsDigit(c))
                {
                    int num_1 = stack.Pop();
                    int num_2 = stack.Pop();
                    switch (c)
                    {
                        case '+':
                            result = num_1 + num_2;
                            break;
                        case '-':
                            result = num_1 - num_2;
                            break;
                        case '*':
                            result = num_1 * num_2;
                            break;
                        case '/':
                            result = num_1 / num_2;
                            break;
                    }
                    stack.Push(result);
                }
                else if (Char.IsDigit(c))
                {
                    stack.Push(int.Parse(c.ToString()));
                }
            }
            Console.WriteLine(stack.Pop());

            Console.ReadKey();
        }
    }
}

            string input = "231*+9-";
            int result = 0;

            Stack<int> stack = new Stack<int>();

            for (int i = 0; i < input.Length; i++)
            {
                char c = input[i];

                if (!Char.IsDigit(c))
                {
                    int number_1 = stack.Pop();
                    int number_2 = stack.Pop();
                    switch (c)
                    {
                        case '+':
                            result = number_2 + number_1;
                            break;
                        case '-':
                            result = number_2 - number_1;
                            break;
                        case '*':
                            result = number_2 * number_1;
                            break;
                        case '/':
                            result = number_2 / number_1;

                    stack.Push(result);
                }
                else if (Char.IsDigit(c))
                {
                    stack.Push(int.Parse(c.ToString()));
                }
            }
            Console.WriteLine(stack.Pop());

    }

中序轉後序

e.g.

2 + 3 * 5 - 9 => 2 3 5 * + 9 -

2 + 3 * 5 - 9 / 3 => 2 3 5 * + 9 3 / -

2 + 3 - 5 => 2 3 + 5 -

這邊是參考文章的, 中序轉後序的文字說明後寫出, 簡單的想法就是

碰到數字就輸出

碰到運算符(operator), +, -, *, / 等等就需要判斷優先順序存到記憶體(stack)

+ 或 -：如果記憶體中有比它優先級高或相等的運算符(如 * 或 /), 則先輸出記憶體中的運算符, 然後將 + 或 - 存入記憶體. 否則，直接將 + 或 - 存入記憶體。

* 或 /：如果記憶體中有比它優先級高或相等的運算符(如 * 或 /), 則先輸出記憶體中的運算符, 然後將 * 或 / 存入記憶體. 否則，直接將 * 或 / 存入記憶體。

namespace <namespace_name>
{
    internal class Program
    {
        static void Main(string[] args)
        {
            string str = "2+3*5-9/3";
            // 235*+93/-

            List<string> output = new List<string>(); 
            Stack<char> stack_symbol = new Stack<char>();

            for (int i = 0; i < str.Length; i++)
            {
                char c = str[i];
                if (Char.IsDigit(c)) // 如果是「數字」 就添加到 List
                {
                    output.Add(c.ToString());
                }
                if (!Char.IsDigit(c))
                {
                    if (stack_symbol.Count == 0) // 如果 stack 裡面沒有任何 operator 就添加至 stack
                    {
                        stack_symbol.Push(c);
                    }
                    else if (stack_symbol.Count != 0) // stack 裡面有 operator 需判斷 operator 的優先級
                    {
                        switch (c)
                        {
                            case '+':
                            case '-':
                                while (stack_symbol.Count > 0)
                                {
                                    output.Add(stack_symbol.Pop().ToString());
                                }
                                stack_symbol.Push(c);
                                break;
                            case '*':
                            case '/':
                                if (stack_symbol.Peek() == '*' || stack_symbol.Peek() == '/')
                                {
                                    output.Add(stack_symbol.Pop().ToString());
                                    stack_symbol.Push(c);
                                }
                                else
                                {
                                    stack_symbol.Push(c);
                                }
                                break;
                        }
                    }
                }
            }
            while (stack_symbol.Count > 0)
            {
                output.Add(stack_symbol.Pop().ToString());
            }
            Console.WriteLine(string.Join("", output));

            Console.ReadKey();
        }
    }
}

namespace <namespace_name>
{
    internal class Program
    {
        static void Main(string[] args)
        {
            // empty
            string str = "2+3*5-9/3";
            // 235*+93/-

            string result = Output(str);
            Console.WriteLine(result);

            Console.ReadKey();
        }

        static int Priority(string str) // 分類 operator 的優先順序
        {
            if (str == "*" || str == "/") return 2;
            else if (str == "+" || str == "-") return 1;
            else if (str == "") return 0;
            return 0;
        }

        static bool CheckPriority(string str1, string str2) // 比較 operator 的優先順序
        {
            int num_1 = Priority(str1);
            int num_2 = Priority(str2);
            if (num_2 > num_1)
            {
                return true;
            }
            return false;
        }

        static string Output(string str)
        {
            string result = "";
            string last_operator = "";
            Stack<string> stack = new Stack<string>();

            for (int i = 0; i < str.Length; i++)
            {
                //string c = str[i].ToString();
                if (Char.IsDigit(str[i]))
                {
                    result += str[i];
                }
                else
                {
                    if (!CheckPriority(last_operator, str[i].ToString())) // 如果 str2 的優先級沒有比較高, 取出所有的 operator
                    {
                        while (stack.Count > 0)
                        {
                            result += stack.Pop();
                        }
                    }
                    stack.Push(str[i].ToString());
                    last_operator = str[i].ToString();
                }
            }
            while (stack.Count > 0)
            {
                result += stack.Pop();
            }

            return result;
        }

    }
}
